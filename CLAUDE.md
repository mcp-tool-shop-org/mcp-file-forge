# mcp-file-forge

## What This Does (Prototype)

⚠️ **Prototype Status** — File generation and template system for MCP.

Generates files from templates, with variable substitution and
multi-file project scaffolding capabilities.

## Features

- Template-based file generation
- Variable interpolation
- Project scaffolding
- Batch file operations
- Custom template support

## Architecture

- Template engine
- Variable resolution
- File I/O abstractions
- MCP tool integration

## Key Notes

- Prototype scaffolding tool
- Template-driven generation
- Supports custom templates
- Subject to design changes
